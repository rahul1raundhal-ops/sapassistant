Auto-save is on

Everything you add — processes, steps, captions, and uploaded screenshots — is automatically saved to this browser's IndexedDB storage. Your data persists across sessions and page reloads on this device.

To use on another device or share with a teammate, export your data as JSON and import it there.

Data management

GitHub Pages — deploy in 5 minutes

  1. Go to github.com → sign in → click New repository
  2. Name it sap-sco-assistant → set to Public → click Create
  3. Click uploading an existing file → drag in this index.html file
  4. Export your data above and drag the processes.json file too
  5. Commit both files
  6. Go to Settings → Pages → Source: Deploy from branch → branch: main → Save
  7. Wait ~60 seconds → your live URL appears: https://yourusername.github.io/sap-sco-assistant

Load data from JSON file (optional)

By default all data is stored in IndexedDB. If you want the GitHub Pages version to load from a JSON file on the server, place processes.json next to index.html — this app auto-fetches it on startup and merges with local storage.
